# LotoAnalise

Aplicativo Android para análise probabilística da Lotofácil, com dashboards, jogos salvos e apoio de IA. Desenvolvido como atividade da disciplina de Desenvolvimento de Apps do IFPR – Telêmaco Borba. Código aberto.

## Visão geral
- **Dashboards probabilísticos**: números mais/menos frequentes, distribuição pares/ímpares, soma média, sequências e repetição de dezenas entre concursos.
- **Sugestão guiada**: gera combinações com base em frequência histórica e permite recalcular variações.
- **Jogos salvos**: armazena, lista e recalcula apostas; chips destacam cada dezena para leitura rápida.
- **Assistente IA**: chat para tirar dúvidas, validar ideias e receber explicações; aceita anexos (prints, tabelas) como contexto.
- **Tema configurável**: visual clássico por padrão e opção neon; admins aplicam o tema para todos os usuários.
- **Contexto local e notificações**: busca lotéricas próximas e oferece notificações in-app.

## Principais fluxos
- **Análises**: acompanhe cards com métricas probabilísticas e veja os últimos concursos em lista com chips.
- **Montagem de jogos**: selecione dezenas, salve, recalcule e reutilize combinações.
- **Interação com IA**: descreva a pergunta (texto/imagem) e receba sugestões ou validações de apostas.
- **Personalização**: escolha de tema feita por administradores e aplicada globalmente.

## Público-alvo
Para quem quer acompanhar Lotofácil de forma estruturada, comparar probabilidades e manter um histórico organizado de jogos.
